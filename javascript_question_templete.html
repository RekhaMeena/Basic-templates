All the Best for your Spy Test!

Welcome to generic website that will seem to ask mundane question to normal users, but you secretly are looking for a spy! Anyone visiting your website will be asked a series of questions, only the Spy will be able to give specific answers you expect. If all questions are answered correctly, you will post a secret message in console for the Spy to read!

Here are the four conditions you will check for the spy:

  • The Spy has the same first letter of her First Name and Last Name
  • The Spy is between the Age of 20 and 30(exclusive of 20 and 30)
  • The Spy is at least 170 centimeters tall
  • The Spy has a pet name that ends with the letter "y"

I would suggest you use Javascript to prompt the user for information and set these to variables. Then use these variables to check for four conditions to all be true. It's up to oyu what the Secret message is!

A spy with the following information should see your secret message:

  • Name of the spy
  • Age
  • Length
  • Pet name

Keep in mind you can ask the question prompts in whatever order you want, you can even separate asking for first name and then another prompt for last name.

Also remember to switch out the script src to your own.js file right now it's connected to .html which contains the solutions.
